# My HTML Project

## Description
A simple HTML webpage created using basic HTML tags.

## Features
- Heading using `<h1>`
- Paragraph using `<p>`
- Simple webpage structure

## Code

```html
<!DOCTYPE html>
<html>
<body>
<h1>what are you looking meri Jano</h1>
<p>Welcome to my world!</p>
</body>
</html>
```

## How to Run
1. Create a file named `index.html`
2. Paste the code into the file
3. Save the file
4. Open it in any web browser

## Output
Displays:
- A heading: **what are you looking meri Jano**
- A paragraph: **Welcome to my world!**

## Technologies Used
- HTML5
